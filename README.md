# 🚀 AI Career Advisor & Skill Gap Analyzer

A lightweight, high-performance web application that recommends top tech roles based on user skillsets and job descriptions.
It uses custom natural language processing (NLP) pipelines built completely from scratch using Python's standard libraries.

## ✨ Features
- **Hybrid Matching Engine:** Uses TF-IDF with Cosine Similarity paired alongside a KNN Jaccard metric.
- **Dynamic Skill Parsing:** Automatically extracts technical keywords from copy-pasted job descriptions.
- **Interactive Dashboard:** Modern CSS layout featuring animated skill match metrics and dynamic documentation links for skill gaps.

## 🛠️ Tech Stack
- **Backend:** Python, Flask
- **Frontend:** HTML5, CSS3 (Modern Variables, Flexbox/Grid), Native JavaScript (ES6)
- **NLP Implementation:** Pure Python (`math`, `re`, `collections.Counter`) — *No heavy libraries like Scikit-Learn or NumPy used.*
